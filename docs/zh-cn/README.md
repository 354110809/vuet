## 简体中文文档


### Vuet是什么？
Vuet是Vue.js的一种状态管理模式，它以`模块化管理`为核心，以`更新规则`为主，`手动更新`为辅来进行模块的状态更新，总之它给你提供了一片草原，由你来决定如何驰骋
